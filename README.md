# Cálculos solar

Esta aplicación fue desarrollada para realizar cálculos para instalaciones solares. La idea es poder realizar cálculos rápidos aproximados para hacer una idea de los materiales y recursos necesarios para una instalación solar fotovoltaica.

Está desarrollada para usarla como un *script* en *Google App Script*. Está escrita en *TypeScript* y usa [ParcelJS](https://parceljs.org/) para el empaquetado.

También requiere una **hoja de cálculo de Google** que se usará a modo de base de datos para los productos que se incorporarán a la instalación.
